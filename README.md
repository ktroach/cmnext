# CMNexT 
Database-less Content Management app in Next.js, minus the headaches

Includes
- NextJS 14 App Router
- ContentLayer
- Shadcn/UI
- Tailwind 

Features
 - Clean Stack 
 - Minimal Boilerplate
 - Functional 
 - Stylish
 - Dead Simple 

1) Clone this repo
2) Install dependencies 
3) Mod and Test 
4) Deploy to Vercel

## Install Dependencies

```
pnpm install
```

```
pnpm run shadcn:add
```

## Build and Run

```
pnpm run build
```

```
pnpm run dev
```

http://localhost:3000/
